

    Значения сеточной функции могут быть прочитаны
из текстового файла,
  в текстовом файле значения должны быть записаны по строкам
 без пробелов, 
в каждой строке сначала значение аргумента,
потом значение функции,
  x  f(x)
в файле не должно быть пустых строк,

   значения аргументов не обязательно должны быть упорядоченны,
при чтении файла выполняется сортировка значений аргументов,
