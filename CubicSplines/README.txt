Значения сеточной функции могут быть прочитаны из текстового файла
В текстовом файле значения должны быть записаны по строкам без пробелов, в каждой строке сначала значение аргумента, потом значение функции, x  y
В файле не должно быть пустых строк
Значения аргументов не обязательно должны быть упорядоченны
При чтении файла выполняется сортировка значений аргументов
